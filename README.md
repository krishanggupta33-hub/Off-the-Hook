Off The Hook 

Off The Hook is a fast-paced, physics-based grappling platformer built with a retro-arcade Tron aesthetic. Swing across massive chasms, build momentum, 
and navigate 10 distinct levels using an infinite-range grappling hook. Built entirely from scratch using HTML5 Canvas and Vanilla JavaScript—no external game engines or physics libraries


Features

1. Custom Physics Engine: Built-in gravity, friction, velocity clamping, and pendulum swing mechanics.

2. Game Feel : Integrated screen shake on heavy impacts, dynamic particle bursts, and a glowing neon trail system.

3. Tron Aesthetic: High-contrast retro pixel art, CRT-style scanlines, and a reactive neon color palette.

4. 10-Level Campaign: Ranging from simple to complex multi-anchor chasm climbs.

5. Mobile & Desktop Support: Fully playable on a keyboard/mouse or via touch controls.

Controls


Desktop


[ A ] / [ D ] : Move Left / Right

[ SPACE ] : Jump / Cut Rope

[ LEFT CLICK ] : Shoot Grapple (Hold to swing)


Mobile


[ ◀ ] / [ ▶ ] : Move Left / Right

[ ▲ ] : Jump / Cut Rope

[ TAP SCREEN ] : Shoot Grapple (Hold to swing)
